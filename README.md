# Interlink Enterprises Portfolio Website

This is a modern, responsive company portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## Features
- Home, About Us, Products, Services, Contact Us pages
- Tailwind CSS for styling
- Framer Motion for animations
- Organized project structure for scalability

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure
- `/src/pages` — Main pages (Home, About, Products, Services, Contact)
- `/src/components` — Reusable UI components
- `/src/layouts` — Layout components
- `/src/styles` — Global and custom styles
- `/src/utils` — Utility functions
- `/src/data` — Static data (if needed)
- `/src/hooks` — Custom React hooks
- `/public/images` — Static assets (logo, images)
